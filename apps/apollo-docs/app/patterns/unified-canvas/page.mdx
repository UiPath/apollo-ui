# Unified canvas

The unified canvas is a cross-surface pattern for visualizing and orchestrating work, agents, and system flows within UiPath.

## Overview

The unified canvas provides a common visual framework for building, viewing, and editing flows. It represents tasks, activities, agents, data paths, and decisions in a single, cohesive space.

It replaces fragmented flow-visualization patterns across Studio, Orchestrator (Maestro), and AI components with one consistent model.

## Anatomy

The unified canvas consists of four core elements:

| Element | Description |
|---|---|
| **Nodes** | Visual units representing tasks, activities, agents, or decisions |
| **Edges** | Connections between nodes representing data or control flow |
| **Handles** | Connection points on nodes where edges attach |
| **Contextual controls** | Actions and tools that appear in context with selected canvas elements |
