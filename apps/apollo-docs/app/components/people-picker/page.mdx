# People picker

The people picker is a form input that allows for the choice of one or more people.

## Usage

The people picker allows users to select individuals or groups from a list of options.

By entering names or email addresses into the input field, users can search for particular individuals or groups. As users type, the people picker offers instant suggestions in a menu.

## Anatomy

| Element | Description |
|---|---|
| **Field label** | Describes the purpose of the input |
| **Input text** | The user's typed search query |
| **Input field** | The text input container |
| **Input chip** | Represents a selected person or group |
| **Menu** | List of matching options for the field's value |

## States

Input field states: default, hover, focus, focus typing, filled/selected, scroll, and error.

### Search results

- **No matching items**: Shown when the search returns no results
- **List of matching items**: Shown as the user types a query
