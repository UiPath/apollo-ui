# Data grid

Data grid is used to organize and display data efficiently. The data grid component allows for customization with additional functionality, as needed by your product's users.

## Usage

Data grid is a table that visualizes a dataset and includes options for filtering, sorting, templates, row selection, row grouping, row pinning, and movable columns. Data grids are often used with dynamic or large data sets.

## Anatomy

Data tables display information in a grid-like format of rows and columns, organized in a way that's easy to scan.

| Element | Description |
|---|---|
| **Table title** | Describes the data shown in the table |
| **General actions** | Actions applicable to the whole table |
| **Action panel** | Search, columns, groups, sort, filter, and action box controls |
| **Column headers** | Labels for each column |
| **Single row** | One record in the data set |
| **Single cell** | One data point within a row |
| **Table footer** | Pagination controls and row count |

## Cell variations

Cells are the building blocks that make up the grid. The type of cell can vary depending on the requirements for the column's contents.

## States

Several states are available to communicate how the user can interact with the data grid and to provide feedback on user actions.

## Behaviors

### Resizing

Columns can be resized by dragging the top right part of the column header.
