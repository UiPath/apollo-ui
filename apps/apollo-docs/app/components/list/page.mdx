# List

List is a grouping of related items. Listbox is a vertical list of interactive elements that users can select from.

## Usage

A list presents a group of related information in a scannable format.

## Anatomy

| Element | Description |
|---|---|
| **List header** | (Optional) Label for the group of list items |
| **List item** | An individual row in the list |
| **Overflow** | Handles truncation or overflow of list items |

## Variations

Our products have four common list patterns:

| Variant | Description |
|---|---|
| **Simple list** | A basic vertical list of items |
| **Accordion list** | A list with expandable/collapsible rows |
| **Multi-line list** | ⚠️ Deprecated |
| **Task list** | A list with hover effects, text truncation, and an action menu icon |

## Appearance

### Leading and trailing content

List items can have **leading content** (avatar, icon, checkbox, or radio) and **trailing content** (icon or notification badge).

## Guidelines

- Use sentence case and begin list items with a capital letter
- Maintain consistency across grouped list items to improve scannability
