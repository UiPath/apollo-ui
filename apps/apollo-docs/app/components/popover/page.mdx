# Popover

A popover is a layer that pops up over all other elements on a page.

## Usage

A popover is a layer that appears above all other content on the page. Only one popover can appear at a time, containing varying text and interactive elements.

Popovers are a base layer in some components, like tooltips, overflow menus, and dropdown menus.

## Anatomy

| Element | Description |
|---|---|
| **UI trigger** | An interactive element that triggers the popover to open on click, hover, or focus |
| **Content area** | An area to place text and interactive elements |
| **Container** | The bounding container for the popover content |
