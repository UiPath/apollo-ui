# Side nav

The side nav is a left-side vertical panel that provides hierarchical navigation links for the application.

## Usage

Use the side nav as the primary navigation structure for multi-section applications. It supports expandable and collapsible sections and can include contextual switching between areas of the product.

## Anatomy

| Element | Description |
|---|---|
| **Chevron** | Toggles the expand and collapse state of a navigation section |
| **Footer items** | Fixed navigation links or actions at the bottom of the panel |
| **Context switcher** | Allows users to switch between different product areas or tenants |
| **Folder tree** | Hierarchical structure for organizing nested navigation links |

## Placement

The side nav is always positioned on the left side of the layout.

## Themes

Supports both **light** and **dark** themes.
