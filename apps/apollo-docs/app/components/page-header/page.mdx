# Page header

Page header provides contextual navigation and key actions at the top of a page section.

## Usage

The page header includes a title, optional subtitle, page filter, supporting actions, and breadcrumbs. It helps users understand their current location, the available actions, and how to navigate the content that follows.

## Anatomy

| Element | Description |
|---|---|
| **Page title** | The main title of the page — should be clear, concise, and descriptive |
| **Badge** | (Optional) Highlights the status or importance of the page; can flag "preview" features |
| **Breadcrumb** | Shows the user's location within the product hierarchy |
| **Global page filter** | (Optional) Allows users to narrow down or customize the main content shown on a page |
| **Action buttons** | Allow performing key actions directly from the page header |
| **More options button** | Opens a menu with additional, secondary or less frequently used actions |

## Themes

The page header supports both **light** and **dark** themes.
