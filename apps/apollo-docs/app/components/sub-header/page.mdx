# Sub-header

The sub-header provides contextual navigation and key actions at the top of a page section. It spans the full viewport width and sits below the global header.

## Usage

Use the sub-header to support navigation within a page or feature area and to surface the most important actions available at the current context.

## Anatomy

| Zone | Elements |
|---|---|
| **Left** | Back button, breadcrumbs |
| **Right** | Version indicator, action buttons, status badge, more options button |

## Placement

The sub-header spans the full viewport width and appears directly below the global header, above the main content area.

## Themes

Supports both **light** and **dark** themes.
