# Global header

The global header is part of the product UI shell — a set of shared components that give all UiPath products a consistent feel. Think of it as the "compass" that helps people get oriented, move around, and find what they need no matter which product they're using.

## Usage

The global header is the main navigation tool across all Automation Cloud products. It provides a single, familiar place for users to:

- Switch quickly between products and services
- Use search to find resources, content, or settings
- Access common actions like notifications, help, and account management

By standardizing these elements, the header reduces the learning curve and helps users stay focused on their work.

## Anatomy

The global header is always visible and follows a left-to-right flow: product context on the left, search and global utilities on the right.

The header bar spans the viewport's full width and is the topmost element in the browser window.

## Themes

The global header supports both **light** and **dark** themes.
