# Portal page loader

The portal page loader is a visual animation used to convey page loading, but does not indicate how long the activity will take to complete.

## Usage

The page loader functions as the initial loading animation that appears when the system lacks information about the page's structure during the loading of portal pages.

Use the portal page loader:

- As the very first loading animation when the system does not yet know the structure of the page
- Exclusively as a full-page animation
- As a standalone animation — it should not be combined with any other UI elements or animations

## Anatomy

| Element | Description |
|---|---|
| **Loading indicator** | The animated icon |
| **Label** | Text describing what is being loaded |

## Appearance

### Labels

The default label is "Loading," but it can be customized to provide additional context about what is being loaded.

## Guidelines

**Use a clear and concise label.** The label should be short, meaningful, and accurate to the pending action. Avoid jargon and technical language unless it's critical to the understanding of the intended audience.
