# Slot

A slot is a placeholder element that enables dynamic content injection into a parent layout component.

## Usage

Slots are used in layout components such as modals, drawers, cards, and page layouts. They define a region where component consumers can insert custom content without modifying the parent component's structure.

Common use cases:

- Injecting custom actions into a modal footer
- Adding custom content to a card body
- Inserting contextual controls into a drawer header

## Anatomy

| Element | Description |
|---|---|
| **Slot container** | The placeholder region that receives injected content; contains no content of its own |
