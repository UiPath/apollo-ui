# Icon button

Icon buttons are simple, icon-only buttons that can be used for actions and in toolbars. They're used for repeated or persistent actions on a page and lack visible labels to simplify the UI.

## Usage

A basic icon button indicates to a user that they can perform an inline action on the page, such as:

- Apply an action to an object (e.g., save, delete, add an item, create a new item)
- Trigger a new element to appear on the page (e.g., reveal an overflow menu)

## Appearance

Icon buttons are buttons without visible text labels by default.

## Size

Icon buttons are available in two sizes:

- **Large**: 40px
- **Small**: 32px

## States

Interactive icon buttons have five possible interaction states: enabled, hover, press, focus, and disabled.

## Guidelines

**Be consistent with icon meaning.** Icons used frequently throughout an application must have a consistent meaning. For example, if a blank-paper icon means "new document" on most screens, don't use the same icon elsewhere to mean "reformat document." Consistency aids users with cognitive disabilities and creates a better experience for all users.
