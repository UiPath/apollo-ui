# Angular Playground

Angular playground for testing Apollo Angular components.

## Setup

This is a minimal Angular application structure. To fully set up the Angular playground:

```bash
# Install dependencies
pnpm install

# Generate angular.json and other Angular-specific files
ng new angular-playground --skip-install --directory .

# Or manually create the required Angular configuration files
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
