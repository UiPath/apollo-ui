# Typography

Apollo Vertex uses three font families, each mapped to a CSS custom property and a Tailwind font utility.

## Font Families

<div className="not-prose my-6 flex flex-col gap-4">
  {fontTokens.map((f) => (
    <div key={f.variable} className="rounded-lg border border-border bg-card p-5">
      <div className="flex flex-col gap-1 mb-4">
        <span className="text-sm font-semibold text-foreground">{f.label}</span>
        <code className="text-xs font-mono text-muted-foreground">{f.variable}</code>
        <code className="text-xs font-mono text-muted-foreground">{f.stack}</code>
      </div>
      <p
        className="text-2xl text-foreground"
        style={{ fontFamily: `var(${f.variable})` }}
      >
        {f.specimen}
      </p>
    </div>
  ))}
</div>
