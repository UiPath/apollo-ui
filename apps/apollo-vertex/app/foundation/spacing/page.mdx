# Spacing

Apollo Vertex uses the standard **Tailwind 4px base unit** (`--spacing: 0.25rem`). Every spacing value is a multiple of this base, giving you a consistent and predictable scale across margin, padding, gap, width, height, and more.

**Base unit:** `1 = 4px = 0.25rem`

## Usage

```tsx
<div className="p-4 gap-6 mt-2" />
```

```css
/* CSS variable */
margin: calc(var(--spacing) * 4); /* 16px */
```
