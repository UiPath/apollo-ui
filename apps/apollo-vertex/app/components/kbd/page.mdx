# Kbd

The keyboard key component is used to display keyboard shortcuts.

    <Kbd>⌘</Kbd>
    <Kbd>K</Kbd>
  </div>

  <div className="flex items-center gap-1">
    <Kbd>Ctrl</Kbd>

  </div>

  <div className="flex items-center gap-1">
    <Kbd>⇧</Kbd>
    <Kbd>⌘</Kbd>
    <Kbd>P</Kbd>
  </div>
</div>

## In Button

<div className="p-4 border rounded-lg mt-4">
