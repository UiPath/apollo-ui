# Theme Toggle

A dropdown menu component that allows users to toggle between light, dark, and system theme preferences.

<div className="p-4 border rounded-lg mt-4">
  <ThemeToggle />
</div>

## Features

The Theme Toggle component provides:

- **Light Mode**: Manual light theme selection
- **Dark Mode**: Manual dark theme selection  
- **System Mode**: Automatically matches the user's operating system preference
- Smooth transitions between themes
- Persistent theme preference (stored in localStorage)
- Icon animations when switching themes

## Installation

```bash
npx shadcn@latest add @uipath/theme-toggle
```

This component requires the following dependencies:

- `next-themes` - Theme management
- `lucide-react` - Icons (Sun and Moon)
- `button` - Button component
- `dropdown-menu` - Dropdown menu component

## Usage

First, ensure your app is wrapped with the `ThemeProvider`:

```tsx
import { ThemeProvider } from "@/components/theme-provider"

export default function RootLayout({ children }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  )
}
```

Then import and use the ThemeToggle component:

```tsx
import { ThemeToggle } from "@/components/ui/theme-toggle"
```

```tsx
<ThemeToggle />
```
