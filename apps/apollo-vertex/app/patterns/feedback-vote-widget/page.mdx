# Feedback Vote Widget

A thumbs up / thumbs down control with an optional comment textarea. Designed to capture user sentiment on AI-generated content and integrate with the UiPath Platform Feedback API via [`@uipath/vs-feedback`](https://github.com/UiPath/vertical-solutions).

</div>

## Features

- **Two-state vote** — Thumbs up / down, with `aria-pressed` reflecting the current selection
- **Optional comment** — Toggle a textarea below the vote buttons for free-form feedback
- **Controlled** — Pass `value` + `onVoteChange` (and optionally `comment` + `onCommentChange`)
- **Sizes** — `default` and `sm`
- **Composes existing primitives** — Built on [Button](/components/button) and [Textarea](/components/textarea)

## Installation

Apollo Vertex components are published to a custom shadcn registry under the `@uipath` namespace. Register the namespace once in your project's `components.json`:

```json
{
  "registries": {
    "@uipath": "https://apollo-vertex.vercel.app/r/{name}.json"
  }
}
```

Then install the component:

```bash
npx shadcn@latest add @uipath/feedback-vote-widget
```

## Usage

### Basic

```tsx
"use client";

import * as React from "react";
import { FeedbackVoteWidget } from "@/components/ui/feedback-vote-widget";

export function MyFeedback() {
  const [vote, setVote] = React.useState<"up" | "down" | null>(null);
  return <FeedbackVoteWidget value={vote} onVoteChange={setVote} />;
}
```

### With comment

```tsx
const [vote, setVote] = React.useState<"up" | "down" | null>(null);
const [comment, setComment] = React.useState("");

<FeedbackVoteWidget
  value={vote}
  onVoteChange={setVote}
  showComment
  comment={comment}
  onCommentChange={setComment}
  commentPlaceholder="What could the AI have done better?"
/>
```

### Wiring to the Platform Feedback API

When using `@uipath/vs-feedback`, the vote translates directly to a `FeedbackDraft.isPositive`. The comment goes into the draft's `comment` field as a JSON string — the exact shape is up to your vertical's `TComment` type.

```tsx
import { client } from "./feedback-client"; // your PlatformFeedbackClient

async function handleSubmit(vote: "up" | "down", comment: string) {
  await client.submit({
    isPositive: vote === "up",
    comment: JSON.stringify({ text: comment }),
    metadata: {
      // your TCustom fields
    },
    traceContext: await resolveTraceContext({
      instanceId,
      folderKey,
      agentId,
    }),
  });
}
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `"up" \| "down" \| null` | — | Current vote state |
| `onVoteChange` | `(vote: "up" \| "down") => void` | — | Called when a vote button is clicked |
| `showComment` | `boolean` | `false` | Render the comment textarea below the buttons |
| `comment` | `string` | — | Comment value (controlled) |
| `onCommentChange` | `(value: string) => void` | — | Called when the textarea changes |
| `commentPlaceholder` | `string` | `"Tell us what could be better (optional)"` | Placeholder for the textarea |
| `disabled` | `boolean` | `false` | Disables vote buttons and the textarea |
| `size` | `"default" \| "sm"` | `"default"` | Visual size of the vote buttons |
| `upLabel` | `string` | `"Vote up"` | Accessible label for the up-vote button |
| `downLabel` | `string` | `"Vote down"` | Accessible label for the down-vote button |
