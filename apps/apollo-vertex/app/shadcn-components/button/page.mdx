# Button

Displays a button or a component that looks like a button.

<div className="p-4 border rounded-lg mt-4 flex flex-wrap gap-2">
  <Button>Default</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="destructive">Destructive</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="ghost">Ghost</Button>
  <Button variant="link">Link</Button>
</div>

## Sizes

<div className="p-4 border rounded-lg mt-4 flex flex-wrap gap-2 items-center">
  <Button size="sm">Small</Button>
  <Button size="default">Default</Button>
  <Button size="lg">Large</Button>
  <Button size="icon"><Mail className="h-4 w-4" /></Button>
</div>

## With Icon

<div className="p-4 border rounded-lg mt-4 flex flex-wrap gap-2">

## Loading

<div className="p-4 border rounded-lg mt-4 flex flex-wrap gap-2">

## Installation

```bash
npx shadcn@latest add @uipath/button
```

## Usage

```tsx
import { Button } from "@/components/ui/button"
```

